# prompts

ループを動かす Claude Code ルーチンのプロンプト。

**これは記録であり、実行されるファイルではない。** ルーチンの実体は claude.ai 側にあり、
ここを編集しても自動では反映されない。変更したら API 経由でルーチンを更新すること。

置いている理由は、リポジトリを見ればループの定義が分かるようにするため。
差分がレビューでき、変更履歴も残る。

| ファイル | 実行 | 役割 |
|---|---|---|
| [`request-generator.md`](request-generator.md) | 毎朝 04:30 JST | デモ用の依頼を Slack に投稿する |
| [`morning-loop.md`](morning-loop.md) | 毎朝 05:00 JST | 修復 → 依頼の取り込み → 実装 PR |

## 必要なコネクタ

claude.ai のコネクタとして接続し、ルーチンに添付する。

- **Slack** — 依頼の読み取り、スレッド返信
- **GitHub** — Issue / PR の作成
- **Sentry** — 本番エラーの取得（`morning-loop` のみ）

**ルーチン作成時のコネクタ構成が固定される。** 後からコネクタを追加しても既存の
ルーチンには反映されないため、作り直しが必要。
